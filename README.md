# 🚗 تطبيق كراء السيارات (Car Rental Web App)

تطبيق ويب حديث وسريع لحجز وكراء السيارات بكل سهولة، مصمم بأحدث تقنيات الويب الأساسية (**HTML5**, **CSS3**, **JavaScript ES6**) دون الحاجة إلى أي مكتبات خارجية.

---

## ✨ المميزات الرئيسية (Features)

- 🔍 **بحث وتصفية فورية:** إمكانية البحث عن السيارات بالاسم أو التصفية حسب الفئة (اقتصادية، عائلية، فاخرة).
- 📱 **تصميم متجاوب بالكامل:** يعمل بكفاءة عالية على جميع الشاشات (الهواتف الذكية، الأجهزة اللوحية، وأجهزة الحاسوب).
- 🛒 **نظام حجز تفاعلي:** نافذة منبثقة مخصصة (Modal) لتسهيل عملية حجز السيارة وإدخال بيانات الزبون.
- ⚡ **سريع وخفيف الوزن:** تم بناء التطبيق بلغات الويب النقية (Pure HTML/CSS/JS) لضمان سرعة التحميل والأداء الفائق.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **HTML5:** لبناء هيكلية الصفحات والعناصر التفاعلية.
- **CSS3:** لتنسيق الواجهة، واستخدام نظام Grid و Flexbox للتجاوب، إضافة للأنيميشن الناعم.
- **JavaScript (Vanilla):** لإدارة التفاعل، فلترة البيانات، والتحكم بالنافذة المنبثقة للحجز.

---

## 🚀 كيفية التشغيل (Getting Started)

1. قم بتحميل أو نسخ ملفات المشروع.
2. افتح ملف `index.html` في أي متصفح (Chrome, Firefox, Edge, Safari).
3. استمتع بتجربة التطبيق مباشرة دون الحاجة لتثبيت برامج أو إعداد خوادم!
